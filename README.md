# GestoPro

Sistema de gestão comercial desenvolvido em Flutter para atender pequenos negócios e MEIs.

## Sobre o projeto

O GestoPro foi criado para facilitar o gerenciamento de pequenas empresas, oferecendo uma interface simples e intuitiva para controlar produtos, estoque, caixa e relatórios.

O objetivo é centralizar as principais operações do negócio em um único sistema.

---

## Funcionalidades

- Login de usuários
- Dashboard com indicadores
- Cadastro de produtos
- Controle de estoque
- Movimentação de entradas e saídas
- Controle de caixa
- Relatórios
- Gerenciamento de usuários
- Scanner de código de barras

## Como executar

### Pré-requisitos

- Flutter SDK
- Dart SDK

### Instalação

```bash
git clone https://github.com/seuusuario/GestoPro.git

cd GestoPro/gestopro

flutter pub get

flutter run
```
